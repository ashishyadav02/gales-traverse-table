# Gales Traverse Table

**Developer : ASHISH KUMAR YADAV**  
**Year : 2025**

This program generates the complete Gales Traverse Table with:
- Whole Circle Bearing (WCB)
- Quadrantal Bearing System (QBS)
- Interior Angles
- Bowditch Correction
- Independent Coordinates

## How to Use

1. Go to **Releases** (right side of this page)
2. Download the file `galestraversetable.exe`
3. Double click the `.exe` file
4. Enter stations data as per instructions on screen
